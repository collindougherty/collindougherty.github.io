The Big Blowouts of 2019

Last updated on Oct 25, 2021
Oregon Pounds Nevada 77-6 | JD Humburg | The Oregonian

Every team experiences humiliating losses from time to time. Unless you’re named Alabama or Ohio State. But some losses are particularly bad. For a fun little experiment, let’s take a look at the worst losses of 2019 by scoring margin, and walk through how this was done in R.

suppressMessages(library(tidyverse))
suppressMessages(library(ggalt))
logs <- suppressMessages(read_csv("badfootballlogs19.csv"))

We first begin by loading the relevant packages and our data. In this case, the data is from Sports Reference.

logs1 <- logs %>% separate(Result, into = c("Outcome", "Score"), sep = " ")
logs1 <- logs1 %>% mutate(Score = gsub("\\(","", Score))
logs1 <- logs1 %>% mutate(Score = gsub("\\)","", Score))
newlogs <- logs1 %>% separate(Score, into = c("TeamScore", "OpponentScore"), sep = "\\-")
newlogs <- newlogs %>% mutate(TeamScore = as.numeric(TeamScore), OpponentScore = as.numeric(OpponentScore))

We then made some edits to our downloaded data so that we can work with it. For example, we split our results column into 2 new columns, calling them “TeamScore” and “OpponentScore”, and then made our data numeric. Next, we will create a new dataframe that finds the worst score differentials of the season.

newlogs <- newlogs %>% mutate(Differential = (TeamScore) - (OpponentScore))
worstgames <- newlogs %>% filter(Differential>64)

We will now go ahead and plot the data, highlighting the blowouts in red and circling them.

ggplot() + geom_point(data=newlogs, aes(x=TeamScore, y=OpponentScore), color = "grey") + geom_point(data=worstgames, aes(x=TeamScore, y=OpponentScore), color = "red") + geom_encircle(data=worstgames, aes(x=TeamScore, y=OpponentScore), s_shape=0, expand = .1,color = "black") + 
  labs(title="A Handful of Teams and Their Sad Saturdays", subtitle="The Worst Blowouts of the 2019 Season", caption = "Source: Sports Reference | By Collin Dougherty")+ 
  theme_minimal() + 
  theme(
    plot.title = element_text(size = 16, face = "bold"),
    axis.title = element_text(size = 8), 
    plot.subtitle = element_text(size=10), 
    panel.grid.minor = element_blank()
    )

Yikes. It seems these teams had reason to hang their heads. Sad Saturday indeed.
